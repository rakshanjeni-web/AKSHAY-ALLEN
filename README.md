# Shopping Website

A modern, responsive shopping website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Product Showcase**: Grid layout displaying products with images, names, and prices
- **Shopping Cart**: Add products to cart with quantity tracking
- **Modern UI**: Clean and professional design with hover effects
- **Easy to Customize**: Simple structure for easy modifications and updates

## Files

- `index.html` - Main HTML file with product structure
- `styles.css` - Complete styling and responsive design
- `script.js` - JavaScript for shopping cart functionality

## How to Use

1. Open `index.html` in your web browser
2. Browse the featured products
3. Click "Add to Cart" to add items to your shopping cart
4. The cart functionality tracks items and quantities
5. Open browser console to view cart details

## Customization

### Adding Products
Edit the product items in `index.html`:
```html
<article class="product-item">
    <img src="path/to/image.jpg" alt="Product Name">
    <h3>Product Name</h3>
    <p>$Price</p>
    <button>Add to Cart</button>
</article>
```

### Changing Colors
Update the color scheme in `styles.css`:
- Header/Footer: `#333`
- Primary Button: `#4CAF50`
- Hover: `#45a049`
- Accent: `#ffc107`
